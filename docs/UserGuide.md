# Manual de usuario - SNP Plantillas Pro

## 1. Uso rapido

1. Abre la app.
2. Escribe en el buscador o filtra por seccion.
3. Pulsa la plantilla que quieras usar.
4. Se copia al portapapeles y, si `Solo copiar` esta desactivado, se pega en la ultima ventana externa capturada.

## 2. Controles principales

- `Solo favoritas`: muestra solo plantillas marcadas como favoritas.
- `Solo copiar`: no pega automaticamente, solo copia.
- `Siempre arriba`: mantiene la app flotante.
- `Gestionar`: abre el editor completo de secciones y plantillas.
- `Importar/Exportar`: carga o guarda configuraciones JSON.

## 3. Atajos

- `1` a `9`: ejecuta las primeras plantillas visibles.
- `Esc` en buscador: limpia el filtro.

## 4. Gestion de plantillas

En `Gestionar` puedes:

- Crear, renombrar y eliminar secciones.
- Crear, editar, duplicar y eliminar plantillas.
- Marcar plantillas como favoritas.
- Mover una plantilla entre secciones al editarla.

## 5. Guino de autor

Haz doble clic en el texto de credito para activar el mensaje oculto del desarrollador.
